# ludata

> 收集各类基础数据，为程序员调用做好准备

# 数据提供方式

- `json`文件
- `sql`文件
- `API`调用

> 如果还有其他需要，请提`issue`

# 数据列表

- 国家 `country`
- 省 `province`
- 市 `city`
- 县 `county`
- 街道 `street`
- 地铁线 `metro-line`
- 地铁站 `metro-station`
- 机场 `airport`
